## 📌 Cài Đặt
### Yêu cầu hệ thống:
- Python 3.x
- Các thư viện cần thiết (được liệt kê trong `requirements.txt`)

### Cài đặt thư viện:
```sh
pip install -r requirements.txt
```
hoặc
```sh
pip3 install -r requirements.txt
```

---

## ▶️ Chạy Chương Trình
```sh
python main.py
```
hoặc
```sh
python3 main.py
```
